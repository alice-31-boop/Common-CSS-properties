Common CSS properties

When making a website...(example)

Property Name Description
color: blue; color Sets the text color.
background-color: lightgrey; background-color Sets the background color of an element.
font-size: 16px; font-size Specifies the font of the text.
text-align: center; text-align Aligns the text horizontally (left, right, center, justify).
margin: 20px; margin Sets the space outside the element. css
padding: 10px; padding Sets the space inside the element (between the content and the border).
